## Onekey
 Steam Depot Manifest Uninstaller

## 使用方法
 先去Release下最新发布，打开使用即可

## 开发
本程序使用Python编程语言开发   
要求环境：   
1.Python 3.10及以上   
2.Windows 10及以上   
3.使用Git进行版本管理   

1.克隆项目到本地 

```
git clone https://github.com/ikunshare/Game-Uninstaller
```

2.安装依赖

```
pip install -r requirements.txt
```
